# Currency Swap Form (Vite + React)

A polished, mobile-first currency/token swap form.

## Features
- Live token prices from `https://interview.switcheo.com/prices.json`
- Token icons from Switcheo token-icons repo
- Searchable token picker
- Live quote + min received (slippage)
- Input validation + mocked balances
- Simulated submit with loading indicator + toast

## Run locally

```bash
npm install
npm run dev
```

Build:

```bash
npm run build
npm run preview
```
